# Python Automation Tools

A collection of Python automation projects for real-world business use cases.

---

## Projects

### 1. Scheduled Web Scraper
Automatically scrapes quotes from a website on a schedule and saves 
them to a new file named with the current date. Runs silently in the 
background using headless Chrome.

### 2. PDF Invoice Generator
Reads customer data from an Excel file and automatically generates 
a separate PDF invoice for each customer. Scales to any number of 
customers without changing the code.
